# Code of conduct

## Our standard

DeepLoop should be a respectful, technically honest place to collaborate.

Examples of expected behavior:

- assume good faith
- give actionable technical feedback
- keep claims truthful about evidence and limitations
- respect different levels of experience
- focus disagreement on ideas, evidence, and implementation

Examples of unacceptable behavior:

- harassment, intimidation, or personal attacks
- discriminatory or demeaning language
- deliberate misrepresentation of results, evidence, or project status
- sharing private or sensitive information without permission
- spamming, trolling, or derailing technical discussion

## Scope

This code of conduct applies to repository issues, pull requests, discussions,
and other project spaces where DeepLoop is represented.

## Enforcement

Maintainers may remove, edit, or reject contributions and participation that
violate this code of conduct.

## Reporting

Report conduct concerns through a private maintainer channel when possible. If a
private channel is not available yet, open a minimal public issue requesting
private follow-up without posting sensitive details.
