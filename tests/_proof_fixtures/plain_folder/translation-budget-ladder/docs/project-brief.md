# Project brief

## Goal

Build a small English <-> Chinese translation project that starts only from this
researcher folder and improves the main metric over the stronger starting
baseline on at least one direction.

## Scope

- Bidirectional translation: English to Chinese and Chinese to English
- Start from a very small model roster and justify any escalation
- Prefer benchmark and slice quality over broad speculative expansion

## Important rule

This folder is researcher input only. Do not place DeepLoop-owned runtime state,
generated configs, build code, or experiment implementation code inside this
folder.
