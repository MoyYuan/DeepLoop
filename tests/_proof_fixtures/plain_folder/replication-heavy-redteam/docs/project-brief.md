# Project brief

## Goal

Run a bounded prompt-defense robustness project from this researcher folder only
and carry it through to a final report without writing DeepLoop runtime state
back into the folder.

## Scope

- Small-model prompt-defense or prompt-filtering ideas
- Bounded adversarial prompt set
- Strong preference for repeatable runs and explicit replication notes

## Important rule

This folder is researcher input only. Do not place DeepLoop-owned runtime state,
generated configs, build code, or experiment implementation code inside this
folder.
